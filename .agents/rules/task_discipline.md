# Task Discipline

## Source of Truth

docs/tasks.md is the only valid task planning source.

---

## Forbidden Behavior

Do not:

- create alternative task lists
- execute tasks not defined in docs/tasks.md
- reorder tasks without justification

---

## Deviation Rule

If tasks must change:

→ update docs/tasks.md  
→ register in docs/decision_log.md  

---

## Execution Rule

Always verify alignment with docs/tasks.md before executing any task.
