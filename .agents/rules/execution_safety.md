# Execution Safety

## Safety Rule

Do not proceed without:

- valid context
- defined task
- clear objective

---

## Validation Requirement

Do not consider a task complete without:

- handoff
- validation
- evidence

---

## Error Handling

If inconsistency is detected:

→ stop execution  
→ report issue  
→ request correction  

---

## Assumption Rule

Do not assume missing information.

Always ask before proceeding.
